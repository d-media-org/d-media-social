# GitHub Repo Ready Bundle

Този пакет е подготвен за ново публично GitHub repo, което после да захрани Publer.

Какво има вътре:
- assets/: PNG файловете за периода 2026-06-11 до 2026-06-30
- publer_bulk_upload_for_github.csv: готов CSV с raw.githubusercontent.com шаблон
- github_repo_media_map.csv: mapping между постовете и файловете

Какво трябва да смениш след създаване на repo:
- YOUR_GITHUB_USER
- YOUR_REPO_NAME

Примерен raw URL:
- https://raw.githubusercontent.com/YOUR_GITHUB_USER/YOUR_REPO_NAME/main/assets/2026-06-11_01.png
